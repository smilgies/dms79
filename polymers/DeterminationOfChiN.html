Determination of the Flory-Huggins parameter cN

A consistent choice of the degree of polymerization N is important for the determination of the Flory-Huggins parameter
c of the system which follows the relation

c = A/T + B
where T is the absolute temperature [K], and A and B are constants to be determined from the mean-field prediction for the order-disorder transition
cODT N = 10.5
If the transition temperature TODT is measured for a series of block copolymers with different N, then the constants
A,B can be fitted to
TODT = {A N} / {10.5 - B N}
and cN of the system of block copolymers is then determined for all temperatures. Note that the values for A and B
depend on the choice of N. Hence, A and B values from literature should only be used, if the authors specify exactly,
how N is defined.

Reference: C.M. Papadakis et al., J. Phys. II France 7, 1829 (1997).
