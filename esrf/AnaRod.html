Detlef 's ANA/ROD page

This page contains some information on the so-called experimental version of the ROD and ANA software packages for surface crystallography originally written by Elias Vlieg. In an effort to make these programs applicable to scattering from molecular layers and adapt scattering geometries to the Toika II surface diffractometer, which combines horizontal six-circle geometry and vertical 2+2 geometry in a single instrument, a programming project within the ESRF Sientific Software group was initiated. Thanks to Olof Svensson, Rainer Wilcke, and Claudio Ferrero for the efficient and pleasant colloaboration on the ANA/ROD project. we are indebted to Elias Vlieg for making the most recent ANA/ROD code available.
